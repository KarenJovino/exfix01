Exercício de Fixação

Tag <p>
A tag p cria parágrafos.
Tag <img>
A tag img é usada para adicionar imagens.
Tag <html>
A tag html inicia e também encerra a página html.
Tag <h1>
A tag h1 insere o título principal da página (título de destaque).
Tag <nav>
A tag nav insere na página html uma seção de navegação contendo vários links nos quais podemos navegar de uma página para a outra.
Tag <input>
A tag input é usada como um espaço para preenchimento do usuário (o chamado formulário), como por exemplo: inserir e-mail e senha.
Tag <style>
A tag style é usada para estilizar a página ou parte dela.
Tag <div>
A tag div é usada como uma forma de juntar elementos em um grupo. É como se através de uma div, eu conseguisse dividir meu código html em seções.
Tag <a>
A tag a é conhecida como âncora. Ela é usada como um elemento de ligação e localização entre as páginas e facilita a navegação.
Tag <ul>
A tag ul é utilizada quando a página html possui uma organização em lista, mas sem obedecer a uma ordem específica. Ex: 1,2,3...
Tag <ol>
A tag ol é usada para quando há intenção de organizar a página html de maneira ordenada em uma lista.
